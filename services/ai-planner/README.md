# AI Planner Service

This service provides intelligent resource placement and decision-making capabilities for the Self-Driving Infrastructure Platform.

## Features

- Resource optimization algorithms
- Cost prediction models
- Provider selection logic
- Multi-objective decision making
- Integration with the core platform

## Architecture

The AI Planner service uses a modular architecture with:
- Decision engines for different types of infrastructure decisions
- Model serving components
- Integration with the core platform via gRPC
- Memory and context management
